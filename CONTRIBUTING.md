# Contributing

## Setup
- Node.js 20.x
- `npm install`
- Copy `.env.example` to `.env.local`

## Commands
- Dev: `npm run dev`
- Dev (app only): `npm run dev:app`
- Lint: `npm run lint`
- Build: `npm run build`

## Branching
- GitHub flow (feature branches + PRs to `main`).

## Pull Requests
- Keep PRs small and focused.
- Require at least 1 approval and passing CI.
- Include risk level and rollback notes in the PR description.

## Commit Style
- Conventional commits optional.

## Release Process
- Manual for now. Document build/deploy steps here when production hosting is defined.
